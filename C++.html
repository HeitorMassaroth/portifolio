C++

C++ é uma linguagem de programação open source orientada a objetos, utilizada tanto como linguagem de máquina quanto para softwares empresariais, acadêmicos, jogos, edição de imagens, fotos e vídeos.

Essas características permitem que ela seja considerada uma linguagem tanto de alto quanto de baixo nível, isso sendo possível pelas bibliotecas e funções que ela possui, o que permite que sistemas desenvolvidos nessa linguagem sejam de alto desempenho, estáveis e seguros. Aprender a programar nessa linguagem é fácil, razão essa que torna essa linguagem ainda mais atrativa e amplamente utilizada desde a década de 80.
